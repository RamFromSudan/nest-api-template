## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository with a few ready-made extras:
- TypeORM with PostgreSQL
- Basic and JWT Authentication
- User registration, login, authentication, and, password reset logic
- Mail service with Nodemailer
- Standard API response format
- Validation pipes

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
